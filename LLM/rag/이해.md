# 연구 동기..
왜 내 연구 동기는 비적합 할까..?


연구는 통제 가능한 변수 + 검증 가능한 가설 위에 서야 한다,..
이 에이전트를 만들어서, 현재의 불편한 정책 개선을 하고자 했지만,
이는 Outcome이 학술 평가가 아니라 정책 낙관에 의존하고 있는 것이라고 함..
(규제가 완화 될 것이라는 보장도 없고 주장이 불충분)

-> 그래서 연구 동기가 약해 보이는 것 같다..
왜 기존 모델로는 불가능하고,
내가 제안하는 방식이 구조적으로 우월한가”
를 보여줘야...


기존 RAG 및 Prompt Optimization 연구는
(1) 신뢰성·안전성·Hallucination을 동시에 고려하는 다차원 보상 구조가 정식 정의되지 않았고,
(2) 이를 자동 최적화하는 체계도 부재하며,
(3) 규제 환경 수준의 안정성을 학술적으로 입증한 사례가 없다.

따라서 본 연구는
RAGAS 기반 신뢰성 지표를 보상 신호로 사용한 Verbal-RL 프롬프트 최적화 체계를 정의하고,
안전성·Faithfulness 향상 효과를 정량적으로 검증한다.

라는 얘기를 그냥 할 게 아니라,
자료 조사를 기반으로 해야 하는 것.


“기존 연구는 이런 걸 안 했고, 그래서 내가 한다”
==> 이거 자료조사로 증명해라

#### RAG 품질 평가 논문들

- Prompt Optimization / OPRO / Reflexion / Verbal RL

- Hallucination / Safety / Trustworthy RAG

- RAGAS / Faithfulness 관련 연구

- 금융·법률 같은 High-risk domain 적용 연구
